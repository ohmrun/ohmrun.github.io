staticisLineTerminator(codepoint:Int):Bool

Returns true if the provided codepoint corresponds to one of the following codepoints: - Unicode Character 'LINE FEED (LF)' (U+000A) - Unicode Character 'CARRIAGE RETURN (CR)' (U+000D) - Unicode Character 'LINE SEPARATOR' (U+2028) - Unicode Character 'PARAGRAPH SEPARATOR' (U+2029)

See also:

  • http://www.ecma-international.org/ecma-262/7.0/index.html#prod-LineTerminator

staticisWordChar(codepoint:Int):Bool

Returns true if the provided code point corresponds to one of the following 63 characters:

abcdefghijklmnopqrstuvwxyz
ABCDEFGHIJKLMNOPQRSTUVWXYZ
0123456789_

THIS IS NOT THE ABSTRACT METHODE DESCRIBED IN THE SPEC!!! (It is just a helper for the allowed characters)

See also:

  • http://www.ecma-international.org/ecma-262/7.0/index.html#sec-runtime-semantics-iswordchar-abstract-operation
