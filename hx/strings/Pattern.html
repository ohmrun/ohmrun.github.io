API Documentation

class Pattern

package hx.strings

@:directlyUsed@immutable@threadSafe

Thread-safe API for regex pattern matching backed by Haxe's EReg class.

UTF8 matching (EReg's 'u' flag) is enabled by default.

See also:

  • http://haxe.org/manual/std-regex.html

inlineremove(str:String):String

If MatchingOption.MATCH_ALL was specified, removes all matches. Otherwise only the first match.


>>> Pattern.compile("[.]"     ).remove("a.b.c") == "ab.c"
>>> Pattern.compile("[.]", "g").remove("a.b.c") == "abc"

inlinereplace(str:String, replaceWith:String):String

If MatchingOption.MATCH_ALL was specified, replaces all matches with replaceWith. Otherwise only the first match.


>>> Pattern.compile("[.]"     ).replace("a.b.c", ":") == "a:b.c"
>>> Pattern.compile("[.]", "g").replace("a.b.c", ":") == "a:b:c"

inlinesplit(str:String):Array<String>

Uses matches as separator to split the string.


>>> Pattern.compile("[.]"     ).split("a.b.c") == [ "a", "b.c" ]
>>> Pattern.compile("[.]", "g").split("a.b.c") == [ "a", "b", "c" ]
