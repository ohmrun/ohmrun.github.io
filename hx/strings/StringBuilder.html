API Documentation

class StringBuilder

package hx.strings

@:directlyUsed

An UTF-8 and fluent API supporting alternative to StringBuf with additional functions such as clear(), insert(), isEmpty()
This implementation tries to avoid the creation of intermediate String objects as much as possible.

Constructor

inlinenew(?initialContent:AnyAsString)

Variables

read onlylength:Int

Important: Invocation may result in temporary string object generation during invocation on platforms except Java.

>>> new StringBuilder("").length                       == 0
>>> new StringBuilder("はい").add("は").add("い").length == 4
>>> new StringBuilder("ab").insert(0, "cd").insert(2, "はい").insertChar(1, 32).insertChar(4, 32).length == 8

Returns:

the length of the string representation of all added items
