enum DeclineSum<E>

package stx.fail

import stx.fail.Decline

@:directlyUsed

Types of ways to decline handling a state. INTERNAL refers to states which are unhandleable and not intended to be caught except to report to the user/developer. Note INTERNAL errors are not really intended for inspection and thus do not have too many functions that apply to them.

EXTERNAL refers to subsystem states which may or may no be true errors, but are intended to signal some information to the program at some point.

Values

INTERNAL(digest:Digest)

Internal (FATAL) error

EXTERNAL(v:E)

Programmer defined error state of type E
